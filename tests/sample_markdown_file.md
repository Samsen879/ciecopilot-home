# Topic: 物理学基础

## Subtopic: 力学基础

### Content
力学是物理学的一个重要分支，研究物体在力的作用下的运动规律。牛顿力学是经典力学的基础，包括牛顿三定律和万有引力定律。力学的基本概念包括位移、速度、加速度、力、质量等。

### Formulas
- $F = ma$
- $v = u + at$
- $s = ut + \frac{1}{2}at^2$
- $v^2 = u^2 + 2as$
- $F = G\frac{m_1 m_2}{r^2}$

### Examples
1. 一个质量为2kg的物体，受到10N的力，求加速度
2. 自由落体运动的计算
3. 圆周运动中的向心力计算
4. 简单机械的力学分析

## Subtopic: 热力学基础

### Content
热力学是研究热现象的物理学分支，主要研究热量传递、温度变化以及物质状态变化的规律。热力学第一定律表明能量守恒，第二定律描述了热力学过程的方向性。

### Formulas
- $Q = mc\Delta T$
- $PV = nRT$
- $\Delta U = Q - W$
- $\eta = 1 - \frac{T_c}{T_h}$

### Examples
1. 计算物体吸收热量后的温度变化
2. 理想气体状态方程的应用
3. 热机效率的计算
4. 热传导过程的分析

## Subtopic: 电磁学基础

### Content
电磁学研究电荷、电场、磁场以及它们之间相互作用的规律。库仑定律描述了静电力，法拉第电磁感应定律揭示了电磁感应现象，麦克斯韦方程组统一了电磁理论。

### Formulas
- $F = k\frac{q_1 q_2}{r^2}$
- $E = \frac{F}{q}$
- $\varepsilon = -\frac{d\Phi_B}{dt}$
- $F = qvB\sin\theta$

### Examples
1. 点电荷间的库仑力计算
2. 电场强度的计算
3. 电磁感应现象的分析
4. 带电粒子在磁场中的运动
